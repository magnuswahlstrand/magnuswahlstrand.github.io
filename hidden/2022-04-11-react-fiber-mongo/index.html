Tiny fullstack app

React, Go and MongoDB

In this post I will build a simple app using a combination of tools and libraries that I’m exploring.

  • React (React-Query, Mantine for UI)
  • Go (Fiber for routing, Qmgo db-driver)
  • MongoDB

The goal is to create a simple User List with basic filtering

Backend

I am using Fiber for routing. It is web framework for Go, inspired by Express and designed for fast development.

We have two endpoints in this app

  • GET /users?q=TEXT - list all users, optional query parameter q for a search that will match any field
    • Note: this is not free text search, as Mongo doesn’t support it in the default configuration
  • POST /users - create a new user

The basics - main.go

The Service exposes two HTTP-handlers ListUsers and AddUsers, and wraps the mongo-client for easy access in the DB-methods.

type Service struct {
    mongo *qmgo.QmgoClient
}

We set up our mongo-client to talk to MongoDB on localhost.

var mongoConfig = &qmgo.Config{
	Uri:      "mongodb://localhost:27017",
	Database: "user-db",
	Coll:     "users",
}

func mongoClient() *qmgo.QmgoClient {
	mongo, err := qmgo.Open(context.Background(), mongoConfig)
	if err != nil {
		log.Fatalln(mongo)
	}
	return mongo
}

Finally, we add a call the mongo setup function, create a new Fiber-router with CORS-middleware and our two handlers, and have it listen on port :8080.

func main() {
	mongo := mongoClient()

	s := Service{
		mongo: mongo,
	}

	app := fiber.New()
	app.Use(cors.New(cors.Config{AllowOrigins: "http://localhost:3000"}))
	app.Get("/users", s.ListUsers)
	app.Post("/users", s.AddUser)

	log.Fatal(app.Listen(":8080"))
}

The handlers

The List handler
type ListUsersResponse struct {
    Users []User `json:"users"`
}

func (s *Service) ListUsers(c *fiber.Ctx) error {
	users, err := s.listJobs(c.Context(), c.Query("q", ""))
	if err != nil {
		return c.Status(fiber.StatusInternalServerError).JSON(fiber.Map{})
	}

	return c.JSON(ListUsersResponse{users})
}
Tying it all together

Finally, we need to setup the MongoDB-client and the Fiber-router.

var mongoConfig = &qmgo.Config{
	Uri:      "mongodb://localhost:27017",
	Database: "user-db",
	Coll:     "users",
}

func mongoClient() *qmgo.QmgoClient {
	ctx := context.Background()
	mongo, err := qmgo.Open(ctx, mongoConfig)
	if err != nil {
		log.Fatalln(mongo)
	}
	return mongo
}
